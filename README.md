# my-dotfiles-repo
This repo uses **GNU Stow** to manage symlinked config under `~`.

│       ├── dev.txt
│       ├── drivers.txt
│       ├── fonts.txt
│       ├── plasma.txt
│       └── xorg.txt
├── README.md
├── starship
│   └── .config
│       ├── .git
│       └── starship.toml
├── tmux
│   └── .config
│       └── tmux
